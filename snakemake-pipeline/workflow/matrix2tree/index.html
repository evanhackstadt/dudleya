Distance Matrix to Tree

Upload a distance-matrix CSV or Excel file and infer NJ or UPGMA trees with validation, outgroup rooting, branch-length handling, and customizable visualization.

Created by Dr. Justen Whittall for BIOL173 Students in Winter 2026

📁 1) Upload

Format: first column is sample names, first row headers are the same sample names in the same order. Diagonal should be 0. Full or triangular matrices are accepted. File types: CSV, XLSX, XLS.

Example matrix format 
Taxon,Sp1,Sp2,Sp3,Sp4
Sp1,0,,,
Sp2,0.12,0,,
Sp3,0.18,0.09,0,
Sp4,0.25,0.14,0.07,0

🧮 2) Inference Knobs

Use Ctrl/Cmd-click to select multiple samples.
Negative branch lengths are automatically redistributed to descendants.
